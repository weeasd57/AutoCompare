üöÄ Installation

1

Extract Files

Extract the downloaded ZIP file to your desired location.

unzip autocompare.zip
cd autocompare
2

Install Dependencies

Run npm install to download all required packages.

npm install
3

Create Database

Create a new MySQL database for the application.

mysql -u root -p
CREATE DATABASE autocompare CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
exit;
4

Import Database Schema

Import the database structure from the included SQL file.

mysql -u root -p autocompare < database.sql

Or use phpMyAdmin to import database.sql

5

Configure Environment

Copy the example environment file and edit with your settings.

cp .env.example .env.local

Edit .env.local with your database credentials (see Configuration section).

6

Run the Application

Start the development server or build for production.

# Development
npm run dev

# Production
npm run build
npm start
7

Setup Admin Account

Visit http://localhost:3000/setup to create your admin account.

‚úÖ Done! Your AutoCompare installation is complete. Visit http://localhost:3000 to see your site.

üí∞ Ads & External APIs

AutoCompare includes optional integrations for monetization and vehicle image sourcing.

Google AdSense

AdSense support is enabled by setting NEXT_PUBLIC_ADSENSE_CLIENT. The root layout will load the AdSense script only when this variable is present.

Note: You must use your own AdSense publisher id and create your own ad units (slot ids).

Example usage (React)

import { GoogleAdSlot } from '@/components/GoogleAdSlot';

export function ExamplePage() {
  return (
    <div>
      <GoogleAdSlot slot="1234567890" />
    </div>
  );
}

Google Image Search API (Custom Search)

The app provides a server API route that queries Google Custom Search (Image) to fetch candidate vehicle images. This is used by admin tooling to suggest and save vehicle images.

  • Endpoint: /api/vehicle-image?make=Toyota&model=Camry&year=2024
  • Required env vars: GOOGLE_API_KEY, GOOGLE_CSE_ID
  • Pagination: use start parameter to fetch multiple results
Security: Keep GOOGLE_API_KEY server-side only. Never expose it to the browser.

‚ùì FAQ

How do I reset my admin password?

Run this SQL query in your database:

UPDATE admins SET password_hash = '$2a$12$LQv3c1yqBWVHxkd0LHAkCOYz6TtxMQJqhN8/X4.VTtYn.Pw5S5KUy' WHERE email = 'your@email.com';
-- This sets the password to: password123

Can I add more vehicle fields?

Yes! Add new columns to the vehicles table in MySQL, then update the TypeScript types in src/types/vehicle.ts.

How do I deploy to production?

Build the application and use a process manager like PM2:

npm run build
pm2 start npm --name "autocompare" -- start

The images are not loading?

Make sure the image_url field contains valid, publicly accessible URLs. For external images, you may need to configure next.config.js to allow the domain.

How do I enable HTTPS?

Use a reverse proxy like Nginx with Let's Encrypt SSL certificates, or deploy to a platform like Vercel that provides HTTPS automatically.
